Cautare binara

Algoritmul de căutare binară este un algoritm eficient pentru căutarea unui element într-un tablou sortat. Principiul său de bază constă în compararea elementului căutat cu elementul din mijloc al tabloului și în continuarea căutării în jumătatea corespunzătoare a tabloului, eliminând astfel jumătatea în care sigur nu se poate afla elementul căutat. Procesul continuă recursiv până când elementul este găsit sau jumătatea de căutare devine vidă. 
            #include iostream
                #include vector
                    
                int binarySearch(const std::vector int & array, int target) {
                    int left = 0;
                    int right = array.size() - 1;
                    
                    while (left <= right) {
                        int mid = left + (right - left) / 2;
                    
                        if (array[mid] == target) {
                            return mid;
                        }
                        else if (array[mid] < target) {
                            left = mid + 1;
                        }
                        else {
                            right = mid - 1;
                        }
                    }
                    
                    return -1;
                }
                    
                int main() {
                    std::vector(int)  vec = {1, 3, 5, 7, 9, 11, 13};
                    
                    int target = 7;
                    int index = binarySearch(vec, target);
                    
                    if (index != -1) {
                        std::cout << "Elementul " << target << " se află la indexul " << index << std::endl;
                    }
                    else {
                        std::cout << "Elementul " << target << " nu a fost găsit în vector." << std::endl;
                    }
                    
                    return 0;
                }
